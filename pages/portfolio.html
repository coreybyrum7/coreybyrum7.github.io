Admin from anywhere with the dynamic Admin menu.
The admin menu is one of my favorite creations. I'll admit, the general concept was poached from the AWS Console but I've made it my own. The administrator options for the Training Management Studio platform by LOD, were previously listed on a singular page. There was no search menu and finding what you were looking for often involved pressing CTRL F to use the browsers find functionality.
Features include:
  • Open from anywhere. The admin menu lives in the site navigation. Only loading when accessed for the first time.
  • Open and type. The search module retains focus upon initially opening the admin menu. Pressing enter while in the search bar selects the first option available. This allows the user to find what they're looking for without ever touching their mouse.
  • History. Your most frequented options are saved in the history section for ease of access.
  • Grouping. The admin menu features two buttons in the top right of the page that allow consumers to group by functionality and alphabetically.
  • Keyboard accessible. Open and utilize the admin menu 100% without touching your mouse.

The LODs website.
The learnondemandsystems.com website was built with WordPress using the Avada theme from the Envato Marketplace. The Avada theme is fantastic. With some additional CSS, there really isn't anything I can't do with this site. The site is fully responsive and is our flagship at LODs.

The 'Netflix' effect with LODs' catalogs.
Seamless, layered UI, allows a user to dive deep into an experience while prioritizing the most important content.
In all LODs' catalog browsers (classes, courses, learning paths, and the store) I've implemented two cards for every entity. The cards load dynamically based on the given filters. The first card is read-only and features a main image, title, author, rating (if available), enrollment status (if available), and the bookmark status (if available).
Hovering over a card will animate the hovered card into a lower level detailed card. The detailed card features more detailed information and provides actionable items, such as starting a class or a course, bookmarking an item, or diving into even further details.
Company Roadmap.
The more you traverse my site, the more you'll know I'm a big fan of WordPress. LODs wanted a way to show customers recently launched items as well as items that were coming down the pipeline. The Roadmap page features a subscribe button that allows a customer to subscribe to Roadmap page updates. Wordpress by default, creates RSS feeds for all blog pages. I created a hook between the marketing platform, ClickDimensions, and these WP blog feeds. When a new post has been added to these specific blog categories, an email containing the updates is sent to subscribed customers.
The roadmap page features an accordion that displays 4 different categories of posts. Our curators create posts for recently launched items, items that are coming soon, and items that have been planned for future development. Each post is displayed as an excerpt within the accordion, a read more button is available for further details. The bottom of the page features the archives. This is simply where all posts go to, LIVE, after they've been deployed for a defined amount of time. The months are collapsible, allowing the user to see more whilst minimizing scrolling.
